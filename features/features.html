Features

  • Kestra
  • Features
  • Features

An orchestrator platform

Build complex pipeline in couple of minutes.

Any kind of flows !

Kestra understand the logic of your flow and handle any workflow with parallel, loop, depending flows, ... System have failures, and let Kestra handle the retry, timeout, ... that could happen.

Any plugins !

Plugins offers deep integration with systems allowing to make complex workflows without relying command line. Use official one or build your own easily!

Cloud native & scalable !

Kestra is build with around top cloud native technologies and a strong architecture! Scale to million of executions without the pain.

An scheduler platform

Launch your flows whatever is your need!

Event based!

Kestra can execute flows based on events coming from core or plugins like :

  • result of others flows in real time
  • detection of file from an SFTP or Google Cloud Storage
  • result of a sql query
  • ...

Time based scheduling!

Or execute at regular interval your flows based on cron expression. You have more complex need, rely on conditions to limit the scheduling of your flows like first month of the month, weekend only, ...

Api or UI based !

Don't find your needs, you can still rely on Kestra API (a simple curl can do the job), webhook to trigger a flow from any application or simply start it directly from the Web UI !

A extensible data orchestrator

Kestra is build with data at heart and integrate directly with your source & destination, no need to choose between ETL or ELT, both are possible.

ETL or ELT!

Never choose again between ELT or ETL! Kestra handle both at scale. Choose the method you need : convert from standard format, use task to transform data directly or load raw data and let the database do the hard work.

Directly connected to systems!

Plugins offers rich integrations with all the underlying systems, get rich outputs, gather metrics for all tasks executed

Cloud native & scalable !

Kestra is build with around top cloud native technologies and a strong architecture! Scale to million of executions without the pain.

An all in one platform

kestra is composed of a rich UI allowing to create, run & monitors all your flows, a full scalable & cloud native architecture and based on plugins allowing you to used or create yours easily.

Rich & realtime UI

Create & Edit your flows directly on your browser, execute and monitor all your executions in realtime!

Low code!

Kestra flow are simple descriptive YAML files. No need to learn another language to create a flow!

Cloud native & Infinitely scalable

Scale to millions of executions without the pain since Kestra is built on cloud native & Scalable solutions.

Easy to extend

Plugins are class first in Kestra, many are available from Kestra core team but you can create one easily.

Ship faster

Save time, complexity, and development costs.

Full platform for end to end workflows

Use all the tools to be able to delivered entreprise grade workflow in autonomy.

Live edit in the browser

No desktop application installation needed, a simple browser to edit & monitor all your flows.

Create your flow with Terraform

Deploy Kestra flows as your cloud dependencies for atomic release process.

An Enterprise Edition

The enterprise edition fit all the company needs : Security & Productivity

SSO/OIDC & RBAC!

Secure your kestra with Single Sign on with your Open Id provider and control fine grained ACL for users & groups.

Audit Log & Secrets!

Integrate your secret manager to never disclose your password & key and audit all the resources created on your instance.

Improuve your productivity!

Configure a namespace with task defaults, variables, secrets to automatically configure all flows and follow them with one eye dashboard!

And many more use developing plugins !

Start building with Kestra and enjoy yours pipelines, the only limit is your imagination.
