    # Equality

    Test if two elements are equals. Usage:

    Render 'yes' or 'no':

      {{#eq a b}}
    yes
  {{else}}
    no
  {{/eq}}

    Render 'true' or 'false':

      {{eq a b}}

    Render 'y' or 'n':

      {{eq a b yes='y' no='n'}}

    # And operator

    And operator. This is a boolean operation.

    Truthiness of arguments is determined by isEmpty, so this helper can be used with non-boolean values.

    Usage:

    Render 'yes' or 'no':

      {{#and a b}}
    yes
  {{else}}
    no
  {{/lt}}

    Multiple arguments are supported too:

      {{#and a b c d}}
    yes
  {{else}}
    no
  {{/lt}}

    Render 'true' or 'false':

      {{and a b}}

    Render 'y' or 'n':

      {{and a b yes='y' no='n'}}

    # Compare operator

    Compare to object as comparables.

      {{cmp a b}}

    Renders 1 if a > b, 0 if a == b -1 if a < b
