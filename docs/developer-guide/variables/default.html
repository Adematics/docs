# Default Variables

The following table lists the configurable Variables of the Kestra.io and their default values.

Parameter Description Default
{{ flow.id }} The name of your flow ID "example: barometre-1 ,each-sequential" Matches alphanumeric (same as [a-zA-Z0-9_])
{{ task.id }} The ID of your task ID "example: task-1-XXXXXX ,task-2-VVVVVVVV" Matches alphanumeric (same as [a-zA-Z0-9_])
{{ taskrun.id }} It's the ID of your execution Task Matches alphanumeric (same as [a-zA-Z0-9_])
{{ taskrun.value }} The result of your task "exemple: 0, 1" Matches numeric (same as [0-9])
{{ taskrun.attemptsCount }} The number of times that the task is launched for a given flow
{{ taskrun.startDate }} Retrieves the launch date of the Task ID from the Flow ID
{{ execution.id }} Return the execution ID of your flow Matches numeric (same as [Z0-9_])
{{ inputs.string }} Set Input data in String format for the flow Matches string (same as [a-zA-Z])
{{ inputs.int }} Set Input data in Int format for the flow Matches string (same as [0-9])
{{ inputs.float }} Set Input data in Float format for the flow
{{ inputs.instant }} Enable instant input
{{ inputs.file.content }} Reading file contents on input for the flow ex yaml,txt,....
{{ inputs.file }} Reading file path on input for the flow ex /tmp/file.txt
{{ inputs.file.uri }} Reading file from URI value for the flow ex http://stock.io/myfile.txt
