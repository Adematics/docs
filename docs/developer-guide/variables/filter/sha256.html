# sha256

The sha256 filter returns the SHA-256 hash of the given UTF-8 String.

{{ "test" | sha256 }}

The above example will output the following:

9f86d081884c7d659a2feaa0c55ad015a3bf4f1b2b0b822cd15d6c15b0f00a08
