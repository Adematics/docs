    # Conditions

    Conditions are use in Triggers & Listeners in order to limit the case this one could happen.
    For example:

    • you can limit the listeners to only specified status
    • you can limit triggers to specified namespace prefix
    • ....

    You can pass a list of conditions and all the conditions must match to enable the current action.

    Available condition are :

    • ExecutionFlowCondition
    • ExecutionNamespaceCondition
    • ExecutionStatusCondition
    • FlowCondition
    • FlowNamespaceCondition
    • VariableCondition
