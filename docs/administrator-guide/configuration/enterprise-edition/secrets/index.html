    # Secret Manager configuration

    Kestra offers integration with secret manager. Secrets will be stored directly on a secret back-end and workers will read them at runtime only (in memory only).

    Secrets can be set using the user interface or via terraform

    For now, we provide integration only using Vault (opens new window), but other options will be added in the future.
