FileTransform 

type: "org.kestra.task.scripts.nashorn.FileTransform"

Transform ion format file from kestra with a nashorn (javascript) script.

# Examples

id: "file_transform"
type: "org.kestra.task.scripts.nashorn.FileTransform"
outputs:
  - out
  - map
script: |
  var Counter = Java.type('org.kestra.core.models.executions.metrics.Counter');
  var File = Java.type('java.io.File');
  var FileOutputStream = Java.type('java.io.FileOutputStream');

  logger.info('executionId: {}', runContext.render('{{ execution.id }}'));
  runContext.metric(Counter.of('total', 666, 'name', 'bla'));
  
  map = {'test': 'here'}
  var tempFile = File.createTempFile("nashorn_", ".out")
  var output = new FileOutputStream(tempFile)
  output.write('555\n666\n'.getBytes())

  out = runContext.putTempFile(tempFile)

# Inputs

# from

  • Type: string
  • Dynamic: ✔️
  • Required: ✔️

Source file URI

The file must an ion file generated by kestra

# script

  • Type: string
  • Dynamic: ❌
  • Required: ❌

A full script

# Outputs

# uri

  • Type: object

URI of a temporary result file

The file will be serialized as ion file.
