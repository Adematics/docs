# CsvWriter

type: "org.kestra.task.serdes.csv.CsvWriter"

Read a java serialized data file and write it to a csv file.

# Inputs

# alwaysDelimitText

  • Type: Boolean
  • Dynamic: ❌
  • Required: ❌

Whether fields should always be delimited using the textDelimiter option.

Default value is false

# charset

  • Type: String
  • Dynamic: ❌
  • Required: ❌

The name of a supported charset

Default value is UTF-8.

# fieldSeparator

  • Type: Character
  • Dynamic: ❌
  • Required: ❌

The field separator character (default: ',' - comma)

# from

  • Type: String
  • Dynamic: ❌
  • Required: ✔️

Source file URI

# header

  • Type: Boolean
  • Dynamic: ❌
  • Required: ❌

Specifies if the first line should be the header (default: false)

# lineDelimiter

  • Type: Character[]
  • Dynamic: ❌
  • Required: ❌

The character used to separate rows

# textDelimiter

  • Type: Character
  • Dynamic: ❌
  • Required: ❌

The text delimiter character (default: '"' - double quotes)

# Outputs

# uri

  • Type: URI

URI of a temporary result file
