# CsvReader

type: "org.kestra.task.serdes.csv.CsvReader"

Read a csv file and write it to a java serialized data file.

# Inputs

# charset

  • Type: String
  • Dynamic: ❌
  • Required: ❌

The name of a supported charset

Default value is UTF-8.

# errorOnDifferentFieldCount

  • Type: Boolean
  • Dynamic: ❌
  • Required: ❌

Specifies if an exception should be thrown, if CSV data contains different field count (default: false)

# fieldSeparator

  • Type: Character
  • Dynamic: ❌
  • Required: ❌

The field separator character (default: ',' - comma)

# from

  • Type: String
  • Dynamic: ✔️
  • Required: ✔️

Source file URI

# header

  • Type: Boolean
  • Dynamic: ❌
  • Required: ❌

Specifies if the first line should be the header (default: false)

# skipEmptyRows

  • Type: Boolean
  • Dynamic: ❌
  • Required: ❌

Specifies if empty rows should be skipped (default: true)

# skipRows

  • Type: Integer
  • Dynamic: ❌
  • Required: ❌

Number of lines to skip at the start of the file

# textDelimiter

  • Type: Character
  • Dynamic: ❌
  • Required: ❌

The text delimiter character (default: '"' - double quotes)

# Outputs

# uri

  • Type: URI

URI of a temporary result file
