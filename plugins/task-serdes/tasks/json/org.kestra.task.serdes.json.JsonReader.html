# JsonReader

type: "org.kestra.task.serdes.json.JsonReader"

Read a json file and write it to a java serialized data file.

# Inputs

# charset

  • Type: String
  • Dynamic: ❌
  • Required: ❌

The name of a supported charset

Default value is UTF-8.

# from

  • Type: String
  • Dynamic: ❌
  • Required: ✔️

Source file URI

# newLine

  • Type: Boolean
  • Dynamic: ❌
  • Required: ❌

Is the file is a json new line (JSON-NL)

Is the file is a json with new line separator Warning, if not, the whole file will loaded in memory and can lead to out of memory!

# Outputs

# uri

  • Type: URI

URI of a temporary result file
