# Trigger

type: "org.kestra.task.fs.sftp.Trigger"

Wait for files on Google cloud storage

# Examples

Wait for a list of file on a GCS bucket and iterate through the files

id: gcs-listen
namespace: org.kestra.tests

tasks:
  - id: each
    type: org.kestra.core.tasks.flows.EachSequential
    tasks:
      - id: return
        type: org.kestra.core.tasks.debugs.Return
        format: "{{taskrun.value}}"
    value: "{{ jq trigger '.files[].uri' true }}"

triggers:
  - id: watch
    type: org.kestra.task.fs.sftp.Trigger
    host: localhost
    port: 6622
    username: foo
    password: pass
    from: "/in/"
    interval: PT10S
    action: MOVE
    moveDirectory: "/archive/"

# Inputs

# action

  • Type: string
  • Dynamic: ✔️
  • Required: ✔️

The action to do on find files

# from

  • Type: string
  • Dynamic: ✔️
  • Required: ✔️

The directory to list

# host

  • Type: string
  • Dynamic: ✔️
  • Required: ❌

Hostname of the remote server

# interval

  • Type: object
  • Dynamic: ❓
  • Required: ❌
  • Default: 60.000000000

The interval between test of triggers

# interval.seconds

  • Type: integer
  • Dynamic: ❓
  • Required: ❌

# keyfile

  • Type: string
  • Dynamic: ✔️
  • Required: ❌

Private keyfile to login on the source server with ssh

Must be the ssh key content, not a path

# moveDirectory

  • Type: string
  • Dynamic: ✔️
  • Required: ❌

The destination directory in case off MOVE

# passphrase

  • Type: string
  • Dynamic: ✔️
  • Required: ❌

Passphrase of the ssh key

# password

  • Type: string
  • Dynamic: ✔️
  • Required: ❌

Password on the remote server

# port

  • Type: string
  • Dynamic: ✔️
  • Required: ❌

Port of the remote server

  • Dynamic: ✔️

# proxyPassword

  • Dynamic: ✔️

# proxyPort
