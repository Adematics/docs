# Download

type: "org.kestra.task.fs.sftp.Download"

Download file from sftp server

# Examples

id: "download"
type: "org.kestra.task.fs.sftp.Download"
host: localhost
port: 6622
username: foo
password: pass
from: "/in/file.txt"

# Inputs

# from

  • Type: String
  • Dynamic: ✔️
  • Required: ❌

The fully-qualified URIs that point to destination path

# host

  • Type: String
  • Dynamic: ✔️
  • Required: ❌

Hostname of the remote server

# keyfile

  • Type: String
  • Dynamic: ✔️
  • Required: ❌

Private keyfile to login on the source server with ssh

Must be the ssh key content, not a path

# passphrase

  • Type: String
  • Dynamic: ✔️
  • Required: ❌

Passphrase of the ssh key

# password

  • Type: String
  • Dynamic: ✔️
  • Required: ❌

Password on the remote server

# port

  • Type: String
  • Dynamic: ✔️
  • Required: ❌

Port of the remote server

# username

  • Type: String
  • Dynamic: ✔️
  • Required: ❌

Username on the remote server

# Outputs

# from

  • Type: URI

The fully-qualified URIs that point to destination path

# to

  • Type: URI

The fully-qualified URIs that point to source data
