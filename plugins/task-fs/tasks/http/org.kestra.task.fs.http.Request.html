# Request

type: "org.kestra.task.fs.http.Request"

Request an http server

This task connects to http server, request the provided url and store the response as output

# Examples

Post request to a webserver

id: "request"
type: "org.kestra.task.fs.http.Request"
uri: "https://server.com/login"
headers: 
  user-agent: "kestra-io"
method: "POST"
formData:
  user: "user"
  password: "pass"

# Inputs

# allowFailed

  • Type: boolean
  • Dynamic: ❌
  • Required: ❌

If true, allow failed response code (response code >=400)

# body

  • Type: String
  • Dynamic: ✔️
  • Required: ❌

The full body as string

# contentType

  • Type: String
  • Dynamic: ✔️
  • Required: ❌

The request content type

# formData

  • Type: Map<String, Object>
  • Dynamic: ❌
  • Required: ❌

The form data to be send

# headers

  • Type: Map<CharSequence, CharSequence>
  • Dynamic: ❌
  • Required: ❌

The header to pass to current request

# method

  • Type: Enum
  • Dynamic: ❌
  • Required: ❌
  • Possible Values:
    • OPTIONS
    • GET
    • HEAD
    • POST
    • PUT
    • DELETE
    • TRACE
    • CONNECT
    • PATCH
    • CUSTOM

The http method to use

# uri

  • Type: String
  • Dynamic: ✔️
  • Required: ❌

The fully-qualified URIs that point to destination http server

# Outputs

# body

  • Type: String

The body of the response

# code

  • Type: Integer

The status code of the response

# headers

  • Type: Map<String>

The headers of the response

# uri

  • Type: URI

The url of the current request
