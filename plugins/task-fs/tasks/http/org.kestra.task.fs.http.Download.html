# Download

type: "org.kestra.task.fs.http.Download"

Download file from http server

This task connects to http server and copy file to kestra file storage

# Examples

id: "download"
type: "org.kestra.task.fs.http.Download"
headers: 
  user-agent: "kestra-io"
uri: "https://server.com/file"

# Inputs

# body

  • Type: String
  • Dynamic: ✔️
  • Required: ❌

The full body as string

# contentType

  • Type: String
  • Dynamic: ✔️
  • Required: ❌

The request content type

# formData

  • Type: Map<String, Object>
  • Dynamic: ❌
  • Required: ❌

The form data to be send

# headers

  • Type: Map<CharSequence, CharSequence>
  • Dynamic: ❌
  • Required: ❌

The header to pass to current request

# method

  • Type: Enum
  • Dynamic: ❌
  • Required: ❌
  • Possible Values:
    • OPTIONS
    • GET
    • HEAD
    • POST
    • PUT
    • DELETE
    • TRACE
    • CONNECT
    • PATCH
    • CUSTOM

The http method to use

# uri

  • Type: String
  • Dynamic: ✔️
  • Required: ❌

The fully-qualified URIs that point to destination http server

# Outputs

# code

  • Type: Integer

The status code of the response

# headers

  • Type: Map<String>

The headers of the response

# uri

  • Type: URI

The url of the downloaded file on kestra storage
