UpdateBucket 

type: "io.kestra.plugin.gcp.gcs.UpdateBucket"

# Examples

Update some bucket labels

id: "update_bucket"
type: "io.kestra.plugin.gcp.gcs.UpdateBucket"
name: "my-bucket"
labels:
  my-label: my-value
