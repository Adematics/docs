# ExecutionStatusCondition

type: "org.kestra.core.models.conditions.types.ExecutionStatusCondition"

Condition based on execution status

# Examples

- conditions:
    - type: org.kestra.core.models.conditions.types.ExecutionStatusCondition
      in:
        - SUCCESS
      notIn: 
        - FAILED
