ExecutionNamespaceCondition 

type: "org.kestra.core.models.conditions.types.ExecutionNamespaceCondition"

Condition for an execution namespace

# Examples

- conditions:
    - type: org.kestra.core.models.conditions.types.ExecutionNamespaceCondition
      namespace: org.kestra.tests
      prefix: true

# Inputs

# namespace

  • Type: string
  • Dynamic: ❓
  • Required: ✔️

The namespace of the flow or the prefix if prefix is true

# prefix

  • Type: boolean
  • Dynamic: ❓
  • Required: ❌

If we must look at the flow namespace by prefix (simple startWith case sensitive)
