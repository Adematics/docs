# FlowNamespaceCondition

type: "org.kestra.core.models.conditions.types.FlowNamespaceCondition"

Condition for a flow namespace

# Examples

- conditions:
    - type: org.kestra.core.models.conditions.types.FlowNamespaceCondition
      namespace: org.kestra.tests
      prefix: true

# Inputs

# namespace

  • Type: string
  • Dynamic: ❓
  • Required: ✔️

The namespace of the flow or the prefix if prefix is true

# prefix

  • Type: boolean
  • Dynamic: ❓
  • Required: ❌

If we must look at the flow namespace by prefix (simple startWith case sensitive)
