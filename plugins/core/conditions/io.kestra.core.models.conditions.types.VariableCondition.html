VariableCondition 

type: "io.kestra.core.models.conditions.types.VariableCondition"

Condition based on variables expression

If the result is an empty string, a string containing only space or false, the condition will be false

# Examples

A condition that will return false for a missing variable

- conditions:
    - type: io.kestra.core.models.conditions.types.VariableCondition
      expression: {{ and unknown }}

# Inputs

# expression

  • Type: string
  • Dynamic: ❓
  • Required: ✔️
  • Min length: 1
