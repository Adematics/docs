# Return

type: "org.kestra.core.tasks.debugs.Return"

Simple debugging task that return a renderer value.

This task is mostly useful for debugging purpose.

This one allow you to see inputs or outputs variables for example, or to debug some templated functions.

# Examples

id: "return"
type: "org.kestra.core.tasks.debugs.Return"
format: "{{task.id}} > {{taskrun.startDate}}"

# Inputs

# format

  • Type: String
  • Dynamic: ✔️
  • Required: ❌

The templatized string to render

# Outputs

# value

  • Type: String

The generated string
