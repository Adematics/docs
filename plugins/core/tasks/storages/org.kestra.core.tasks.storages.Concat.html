# Concat

type: "org.kestra.core.tasks.storages.Concat"

Concat files from internal storage.

# Examples

Concat 2 files with a custom separator

id: "concat"
type: "org.kestra.core.tasks.storages.Concat"
files: 
  - "kestra://long/url/file1.txt"
  - "kestra://long/url/file2.txt"
separator: "\n"

Concat file generated by a each tasks

tasks:
  - id: each
    type: org.kestra.core.tasks.flows.EachSequential
    tasks:
      - id: start_api_call
        type: org.kestra.core.tasks.scripts.Bash
        commands:
          - echo {{ taskrun.value }} > {{ temp.generated }}
        files:
          - generated
    value: '["value1", "value2", "value3"]'
  - id: concat
    type: org.kestra.core.tasks.storages.Concat
    files:
      - "{{ outputs.start_api_call.value1.files.generated }}"
      - "{{ outputs.start_api_call.value2.files.generated }}"
      - "{{ outputs.start_api_call.value3.files.generated }}"

# Inputs

# files

  • Type: List<String>
  • Dynamic: ✔️
  • Required: ❌

List of files to be concatenated.

Must be a kestra:// storage url

# separator

  • Type: String
  • Dynamic: ❌
  • Required: ❌

The separator to used between files, default is no seprator

# Outputs

# uri

  • Type: URI

The concatenate file uri.
