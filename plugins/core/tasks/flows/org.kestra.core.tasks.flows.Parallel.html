# Parallel

type: "org.kestra.core.tasks.flows.Parallel"

Process task in parallel

This task processes tasks in parallel. It makes it convinient to process many tasks at once.

# Examples

id: parallel
namespace: org.kestra.tests

tasks:
  - id: parallel
    type: org.kestra.core.tasks.flows.Parallel
    tasks:
      - id: 1st
        type: org.kestra.core.tasks.debugs.Return
        format: "{{task.id}} > {{taskrun.startDate}}"
      - id: 2nd
        type: org.kestra.core.tasks.debugs.Return
        format: "{{task.id}} > {{taskrun.id}}"
  - id: last
    type: org.kestra.core.tasks.debugs.Return
    format: "{{task.id}} > {{taskrun.startDate}}"
