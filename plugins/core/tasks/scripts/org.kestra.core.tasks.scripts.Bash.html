# Bash

type: "org.kestra.core.tasks.scripts.Bash"

Execute a Bash script, command or set of commands.

# Examples

id: "bash"
type: "org.kestra.core.tasks.scripts.Bash"
commands:
- echo "The current execution is : {{execution.id}}"

# Inputs

# commands

  • Type: String[]
  • Dynamic: ✔️

The commands to run

All command will be launched with /bin/sh -c "commands"

# Outputs

# exitCode

  • Type: Int

The exit code of the whole execution

# stdErr

  • Type: List<String>

The standard error of the commands

# stdOut

  • Type: List<String>

The standard output of the commands
