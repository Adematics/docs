Upload 

type: "io.kestra.plugin.aws.s3.Upload"

Upload a file to a S3 bucket.

# Examples

id: "upload"
type: "io.kestra.plugin.aws.s3.Upload"
from: "{{ inputs.file }}"
bucket: "my-bucket"
key: "path/to/file"

# Inputs

# accessKeyId

  • Type: string
  • Dynamic: ✔️
  • Required: ❌

The Access Key Id in order to connect to AWS

If no connection is defined, we will use default DefaultCredentialsProvider that will try to guess the value

# bucket

  • Type: string
  • Dynamic: ✔️
  • Required: ❌

The bucket where to upload the file

# from

  • Type: string
  • Dynamic: ✔️
  • Required: ❌

The file to upload

# key

  • Type: string
  • Dynamic: ✔️
  • Required: ❌

The key where to upload the file

# metadata

  • Type: object
  • Dynamic: ❌
  • Required: ❌

A map of metadata to store with the object in S3.

# region

  • Type: string
  • Dynamic: ✔️
  • Required: ❌

The AWS region to used

# requestPayer

  • Type: string
  • Dynamic: ✔️
  • Required: ❌

Sets the value of the RequestPayer property for this object.

# secretKeyId

  • Type: string
  • Dynamic: ✔️
  • Required: ❌

The Secret Key Id in order to connect to AWS

If no connection is defined, we will use default DefaultCredentialsProvider that will try to guess the value

# storageClass

  • Type: string
  • Dynamic: ✔️
  • Required: ❌

If you don't specify, S3 Standard is the default storage class. Amazon S3 supports other storage classes.

# Outputs

# bucket

  • Type: string

# key

  • Type: string

# versionId

  • Type: string

Version of the object.
