# Download

type: "org.kestra.task.aws.s3.Download"

Download a file to a S3 bucket.

# Examples

id: "download"
type: "org.kestra.task.aws.s3.Download"
bucket: "my-bucket"
key: "path/to/file"

# Inputs

# accessKeyId

  • Type: string
  • Dynamic: ✔️
  • Required: ❌

The Access Key Id in order to connect to AWS

If no connection is defined, we will use default DefaultCredentialsProvider that will try to guess the value

# bucket

  • Type: string
  • Dynamic: ✔️
  • Required: ❌

The bucket where to download the file

# key

  • Type: string
  • Dynamic: ✔️
  • Required: ❌

The key where to download the file

# region

  • Type: string
  • Dynamic: ✔️
  • Required: ❌

The AWS region to used

# requestPayer

  • Type: string
  • Dynamic: ✔️
  • Required: ❌

Sets the value of the RequestPayer property for this object.

# secretKeyId

  • Type: string
  • Dynamic: ✔️
  • Required: ❌

The Secret Key Id in order to connect to AWS

If no connection is defined, we will use default DefaultCredentialsProvider that will try to guess the value

# versionId

  • Type: string
  • Dynamic: ✔️
  • Required: ❌

VersionId used to reference a specific version of the object.

# Outputs

# contentLength

  • Type: integer

Size of the body in bytes.

# contentType

  • Type: string

A standard MIME type describing the format of the object data.

# metadata

  • Type: object

A map of metadata to store with the object in S3.

# uri

  • Type: object

# versionId

  • Type: string

Version of the object.
