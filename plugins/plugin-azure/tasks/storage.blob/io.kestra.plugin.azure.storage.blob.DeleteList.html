    DeleteList 

    type: "io.kestra.plugin.azure.storage.blob.DeleteList"

    Delete a list of key on a Azure Blob Storage.

    # Examples

    id: "delete_list"
type: "io.kestra.plugin.azure.storage.blob.DeleteList"
endpoint: "https://yourblob.blob.core.windows.net"
connectionString: "DefaultEndpointsProtocol=...=="
container: "mydata"
name: "myblob"
prefix: "sub-dir"
delimiter: "/"

    # Properties

    # concurrent

    • Type: integer
    • Dynamic: ❌
    • Required: ❌
    • Minimum: &gt;= 2

    Number of concurrent parallels deletion

    # connectionString

    • Type: string
    • Dynamic: ✔️
    • Required: ❌

    Connection string of the storage account.

    # container

    • Type: string
    • Dynamic: ✔️
    • Required: ❌

    The blob container

    # delimiter

    • Type: string
    • Dynamic: ✔️
    • Required: ❌

    The delimiter for blob hierarchy, "/" for hierarchy based on directories.

    # endpoint

    • Type: string
    • Dynamic: ✔️
    • Required: ❌

    The blob service endpoint.

    additionally parses it for information (SAS token)

    # errorOnEmpty

    • Type: boolean
    • Dynamic: ✔️
    • Required: ❌
    • Default: false

    raise an error if the file is not found

    # filter

    • Type: string

    • Dynamic: ❓

    • Required: ❌

    • Default: FILES

    • Possible Values:

      • FILES
      • DIRECTORY
      • BOTH

    The filter files or directory

    # prefix

    • Type: string
    • Dynamic: ✔️
    • Required: ❌

    Limits the response to keys that begin with the specified prefix.

    # regexp

    • Type: string
    • Dynamic: ✔️
    • Required: ❌

    A regexp to filter on full key

    ex:
    regExp: .* to match all files
    regExp: .*2020-01-0.\\.csv to match files between 01 and 09 of january ending with .csv

    # sasToken

    • Type: string
    • Dynamic: ✔️
    • Required: ❌

    The SAS token to use for authenticating requests.

    This string should only be the query parameters (with or without a leading '?') and not a full url.

    # sharedKeyAccountAccessKey

    • Type: string
    • Dynamic: ✔️
    • Required: ❌

    Shared Key access key for authenticating requests

    # sharedKeyAccountName

    • Type: string
    • Dynamic: ✔️
    • Required: ❌

    Shared Key account name for authenticating requests

    # Outputs

    # count

    • Type: integer
    • Default: 0

    The count of blobs deleted

    # size

    • Type: integer
    • Default: 0

    The size of all blobs deleted
