    ReadRange 

    type: "io.kestra.plugin.googleworkspace.sheets.ReadRange"

    Read a range from a google Sheets

    # Examples

    id: "read_range"
type: "io.kestra.plugin.googleworkspace.sheets.ReadRange"
spreadsheetId: "1Dkd3W0OQo-wxz1rrORLP7YGSj6EBLEg74fiTdbJUIQE"
range: "Second One!A1:I"
store: true
valueRender: FORMATTED_VALUE

    # Properties

    # dateTimeRender

    • Type: string

    • Dynamic: ❌

    • Required: ✔️

    • Possible Values:

      • SERIAL_NUMBER
      • FORMATTED_STRING

    How dates, times, and durations should be represented in the output.

    his is ignored if valueRender is FORMATTED_VALUE.
    More details here (opens new window)

    # fetch

    • Type: boolean
    • Dynamic: ❌
    • Required: ❌

    Whether to Fetch the data from the query result to the task output

    # header

    • Type: boolean
    • Dynamic: ❓
    • Required: ❌

    Specifies if the first line should be the header (default: false)

    # range

    • Type: string
    • Dynamic: ❓
    • Required: ❌

    The range to select

    # scopes

    • Type: array
    • SubType: string
    • Dynamic: ✔️
    • Required: ❌
    • Default: [https://www.googleapis.com/auth/spreadsheets]

    The GCP scopes to used

    # serviceAccount

    • Type: string
    • Dynamic: ❓
    • Required: ❌

    The GCP service account key

    # spreadsheetId

    • Type: string
    • Dynamic: ✔️
    • Required: ✔️

    The spreadsheet unique id

    # store

    • Type: boolean
    • Dynamic: ❌
    • Required: ❌
    • Default: true

    Whether to store the data from the query result into an ion serialized data file

    # valueRender

    • Type: string

    • Dynamic: ❌

    • Required: ✔️

    • Default: UNFORMATTED_VALUE

    • Possible Values:

      • FORMATTED_VALUE
      • UNFORMATTED_VALUE
      • FORMULA

    Determines how values should be rendered in the output.

    More details here (opens new window)

    # Outputs

    # rows

    • Type: array

    List containing the fetched data

    Only populated if 'fetch' parameter is set to true.

    # size

    • Type: integer
    • Default: 0

    The size of the rows fetch

    # uri

    • Type: string

    The uri of store result

    Only populated if 'store' is set to true.
