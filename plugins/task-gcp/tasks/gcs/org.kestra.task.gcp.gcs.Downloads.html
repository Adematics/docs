# Downloads

type: "org.kestra.task.gcp.gcs.Downloads"

Download multiple files from a GCS bucket.

# Examples

Download a list of files and move it to an archive folders

id: "downloads"
type: "org.kestra.task.gcp.gcs.Downloads"
from: gs://my-bucket/kestra/files/
action: MOVE
moveDirectory: gs://my-bucket/kestra/archive/

# Inputs

# action

  • Type: string
  • Dynamic: ✔️
  • Required: ❌

The action to do on find files

Can be null, in this case no action is perform

# allVersions

  • Type: boolean
  • Dynamic: ✔️
  • Required: ❌

If set to true, lists all versions of a blob. The default is false.

# filter

  • Type: string
  • Dynamic: ❓
  • Required: ❌
  • Default: BOTH

The filter files or directory

# from

  • Type: string
  • Dynamic: ✔️
  • Required: ✔️

The directory to list

# listingType

  • Type: string
  • Dynamic: ❓
  • Required: ❌
  • Default: DIRECTORY

The listing type you want (like directory or recursive)

# moveDirectory

  • Type: string
  • Dynamic: ✔️
  • Required: ❌

The destination directory in case off MOVE

# projectId

  • Type: string
  • Dynamic: ✔️
  • Required: ❌

The GCP project id

# regExp

  • Type: string
  • Dynamic: ✔️
  • Required: ❌

A regexp to filter on full path

# Outputs

# blobs

  • Type: array
  • SubType: object

The bucket of the downloaded file
