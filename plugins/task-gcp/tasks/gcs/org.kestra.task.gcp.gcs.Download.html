# Download

type: "org.kestra.task.gcp.gcs.Download"

Download a file to a GCS bucket.

# Examples

id: "download"
type: "org.kestra.task.gcp.gcs.Download"
from: "gs://my_bucket/dir/file.csv"

# Inputs

# from

  • Type: String
  • Dynamic: ✔️
  • Required: ❌

The file to copy

# projectId

  • Type: String
  • Dynamic: ✔️
  • Required: ❌

The GCP project id

# Outputs

# bucket

  • Type: String

The bucket of the downloaded file

# path

  • Type: String

The path on the bucket of the downloaded file

# uri

  • Type: URI

The url of the downloaded file on kestra storage
