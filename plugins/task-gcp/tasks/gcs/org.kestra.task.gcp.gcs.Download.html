# Download

type: "org.kestra.task.gcp.gcs.Download"

Download a file from a GCS bucket.

# Examples

id: "download"
type: "org.kestra.task.gcp.gcs.Download"
from: "gs://my_bucket/dir/file.csv"

# Inputs

# from

  • Type: string
  • Dynamic: ✔️
  • Required: ❌

The file to copy

# projectId

  • Type: string
  • Dynamic: ✔️
  • Required: ❌

The GCP project id

# Outputs

# bucket

  • Type: string

The bucket of the downloaded file

# path

  • Type: string

The path on the bucket of the downloaded file

# uri

  • Type: object

The url of the downloaded file on kestra storage
