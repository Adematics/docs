# List

type: "org.kestra.task.gcp.gcs.List"

List file on a GCS bucket.

# Examples

List files in a bucket

id: "list"
type: "org.kestra.task.gcp.gcs.List"
from: "gs://my_bucket/dir/"

# Inputs

# allVersions

  • Type: Boolean
  • Dynamic: ✔️
  • Required: ❌

If set to true, lists all versions of a blob. The default is false.

# filter

  • Type: Filter
  • Dynamic: ❌
  • Required: ❌
  • Possible Values:
    • FILES
    • DIRECTORY
    • BOTH

The filter files or directory

# from

  • Type: String
  • Dynamic: ✔️
  • Required: ✔️

The directory to list

# listingType

  • Type: ListingType
  • Dynamic: ❌
  • Required: ❌
  • Possible Values:
    • RECURSIVE
    • DIRECTORY

The listing type you want (like directory or recursive)

# projectId

  • Type: String
  • Dynamic: ✔️
  • Required: ❌

The GCP project id

# regExp

  • Type: String
  • Dynamic: ❌
  • Required: ❌

A regexp to filter on full path

# Outputs

# blobs

  • Type: List<org.kestra.task.gcp.gcs.models.Blob>

The list of blobs
