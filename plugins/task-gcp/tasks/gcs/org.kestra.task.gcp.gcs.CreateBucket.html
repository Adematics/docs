# CreateBucket

type: "org.kestra.task.gcp.gcs.CreateBucket"

Create a bucket or update if it already exists.

# Examples

Create a new bucket with some options

id: "create_bucket"
type: "org.kestra.task.gcp.gcs.CreateBucket"
name: "my-bucket"
versioningEnabled: true
labels: 
  my-label: my-value
