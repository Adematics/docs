# Upload

type: "org.kestra.task.gcp.gcs.Upload"

Upload a file to a GCS bucket.

# Examples

id: "upload"
type: "org.kestra.task.gcp.gcs.Upload"
from: "{{ inputs.file }}"
to: "gs://my_bucket/dir/file.csv"

# Inputs

# from

  • Type: String
  • Dynamic: ✔️
  • Required: ❌

The file to copy

# projectId

  • Type: String
  • Dynamic: ✔️
  • Required: ❌

The GCP project id

# to

  • Type: String
  • Dynamic: ✔️
  • Required: ❌

The destination path
