# Copy

type: "org.kestra.task.gcp.gcs.Copy"

Copy a file between bucket

Copy the file between Internal Storage or Google Cloud Storage file

# Inputs

# delete

  • Type: Boolean
  • Dynamic: ❌

Delete the from files on success copy

# from

  • Type: String
  • Dynamic: ✔️

The file to copy

# projectId

  • Type: String
  • Dynamic: ✔️

The google project id to use

# to

  • Type: String
  • Dynamic: ✔️

The destination path

# Outputs

# uri

  • Type: URI

The destination full uri

The full url will be in like gs://{bucket}/{path}/{file}
