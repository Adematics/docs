# SlackExecution

type: "org.kestra.task.notifications.slack.SlackExecution"

Task to send a slack message with execution information

Main execution information are provided in the sent message (id, namespace, flow, state, duration, start date ...).

# Examples

Send a slack notification on failed flow

id: mail
namespace: org.kestra.tests

listeners:
  - conditions:
      - type: org.kestra.core.models.listeners.types.ExecutionStatusCondition
        in:
          - FAILED
  - tasks:
      - id: slack
        type: org.kestra.task.notifications.slack.SlackExecution
        url: "https://hooks.slack.com/services/T00000000/B00000000/XXXXXXXXXXXXXXXXXXXXXXXX"
        channel: "#random"


tasks:
  - id: ok
    type: org.kestra.core.tasks.debugs.Return
    format: "{{task.id}} > {{taskrun.startDate}}"
