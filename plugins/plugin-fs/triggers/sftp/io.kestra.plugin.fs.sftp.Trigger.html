Trigger 

type: "io.kestra.plugin.fs.sftp.Trigger"

Wait for files on a SFTP server

# Examples

Wait for a list of file on a SFTP server and iterate through the files

id: gcs-listen
namespace: io.kestra.tests

tasks:
  - id: each
    type: io.kestra.core.tasks.flows.EachSequential
    tasks:
      - id: return
        type: io.kestra.core.tasks.debugs.Return
        format: "{{taskrun.value}}"
    value: "{{ jq trigger '.files[].path' }}"

triggers:
  - id: watch
    type: io.kestra.plugin.fs.sftp.Trigger
    host: localhost
    port: 6622
    username: foo
    password: pass
    from: "/in/"
    interval: PT10S
    action: MOVE
    moveDirectory: "/archive/"

# Inputs

# action

  • Type: string

  • Dynamic: ✔️

  • Required: ✔️

  • Possible Values:

    • MOVE
    • DELETE

The action to do on find files

# from

  • Type: string
  • Dynamic: ✔️
  • Required: ✔️

The directory to list

# host

  • Type: string
  • Dynamic: ✔️
  • Required: ❌

Hostname of the remote server

# interval

  • Type: object
  • Dynamic: ❓
  • Required: ❌
  • Default: 60.000000000

The interval between test of triggers

# interval.seconds

  • Type: integer
  • Dynamic: ❓
  • Required: ❌

# keyfile

  • Type: string
  • Dynamic: ✔️
  • Required: ❌

Private keyfile to login on the source server with ssh

Must be the ssh key content, not a path

# moveDirectory

  • Type: string
  • Dynamic: ✔️
  • Required: ❌

The destination directory in case off MOVE

# passphrase

  • Type: string
  • Dynamic: ✔️
  • Required: ❌

Passphrase of the ssh key

# password

  • Type: string
  • Dynamic: ✔️
  • Required: ❌

Password on the remote server

# port

  • Type: string
  • Dynamic: ✔️
  • Required: ❌

Port of the remote server

# proxyHost

  • Type: string
  • Dynamic: ✔️
  • Required: ❌

Sftp proxy host

# proxyPassword

  • Type: string
  • Dynamic: ✔️
  • Required: ❌

Sftp proxy password

# proxyPort

  • Type: integer
  • Dynamic: ✔️
  • Required: ❌

Sftp proxy port

# proxyType

  • Type: string
  • Dynamic: ✔️
  • Required: ❌

Sftp proxy type

# proxyUser

  • Type: string
  • Dynamic: ✔️
  • Required: ❌

Sftp proxy user

# regExp

  • Type: string
  • Dynamic: ✔️
  • Required: ❌

A regexp to filter on full path

# rootDir

  • Type: boolean
  • Dynamic: ✔️
  • Required: ❌
  • Default: true

Is path is relative to root dir

# username

  • Type: string
  • Dynamic: ✔️
  • Required: ❌

Username on the remote server
