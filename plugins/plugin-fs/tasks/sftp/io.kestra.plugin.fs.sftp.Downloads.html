    Downloads 

    type: "io.kestra.plugin.fs.sftp.Downloads"

    Download multiple files from sftp server

    # Examples

    Download a list of files and move it to an archive folders

    id: "downloads"
type: "io.kestra.plugin.fs.sftp.Downloads"
host: localhost
port: 6622
username: foo
password: pass
from: "/in/"
interval: PT10S
action: MOVE
moveDirectory: "/archive/"

    # Properties

    # action

    • Type: string

    • Dynamic: ✔️

    • Required: ❌

    • Possible Values:

      • MOVE
      • DELETE

    The action to do on find files

    # from

    • Type: string
    • Dynamic: ✔️
    • Required: ✔️

    The directory to list

    # host

    • Type: string
    • Dynamic: ✔️
    • Required: ✔️

    Hostname of the remote server

    # keyfile

    • Type: string
    • Dynamic: ✔️
    • Required: ❌

    Private keyfile to login on the source server with ssh

    Must be the ssh key content, not a path

    # moveDirectory

    • Type: string
    • Dynamic: ✔️
    • Required: ❌

    The destination directory in case off MOVE

    # passphrase

    • Type: string
    • Dynamic: ✔️
    • Required: ❌

    Passphrase of the ssh key

    # password

    • Type: string
    • Dynamic: ✔️
    • Required: ❌

    Password on the remote server

    # port

    • Type: string
    • Dynamic: ✔️
    • Required: ❌
    • Default: 22

    Port of the remote server

    # proxyHost

    • Type: string
    • Dynamic: ✔️
    • Required: ❌

    Sftp proxy host

    # proxyPassword

    • Type: string
    • Dynamic: ✔️
    • Required: ❌

    Sftp proxy password

    # proxyPort

    • Type: integer
    • Dynamic: ✔️
    • Required: ❌

    Sftp proxy port

    # proxyType

    • Type: string
    • Dynamic: ✔️
    • Required: ❌

    Sftp proxy type

    # proxyUser

    • Type: string
    • Dynamic: ✔️
    • Required: ❌

    Sftp proxy user

    # regExp

    • Type: string
    • Dynamic: ✔️
    • Required: ❌

    A regexp to filter on full path

    # rootDir

    • Type: boolean
    • Dynamic: ✔️
    • Required: ❌
    • Default: true

    Is path is relative to root dir

    # username

    • Type: string
    • Dynamic: ✔️
    • Required: ❌

    Username on the remote server
