Downloads 

type: "io.kestra.plugin.fs.sftp.Downloads"

Download multiple files from sftp server

# Examples

Download a list of files and move it to an archive folders

id: "downloads"
type: "io.kestra.plugin.fs.sftp.Downloads"
host: localhost
port: 6622
username: foo
password: pass
from: "/in/"
interval: PT10S
action: MOVE
moveDirectory: "/archive/"
