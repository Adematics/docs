    Upload 

    type: "io.kestra.plugin.fs.sftp.Upload"

    Upload a file to a sftp server

    # Examples

    id: "upload"
type: "io.kestra.plugin.fs.sftp.Upload"
host: localhost
port: 6622
username: foo
password: pass
from: "{{ outputs.taskid.uri }}"
to: "/upload/dir2/file.txt"

    # Properties

    # from

    • Type: string
    • Dynamic: ✔️
    • Required: ❌

    The file path to copy

    # host

    • Type: string
    • Dynamic: ✔️
    • Required: ✔️

    Hostname of the remote server

    # keyfile

    • Type: string
    • Dynamic: ✔️
    • Required: ❌

    Private keyfile to login on the source server with ssh

    Must be the ssh key content, not a path

    # passphrase

    • Type: string
    • Dynamic: ✔️
    • Required: ❌

    Passphrase of the ssh key

    # password

    • Type: string
    • Dynamic: ✔️
    • Required: ❌

    Password on the remote server

    # port

    • Type: string
    • Dynamic: ✔️
    • Required: ❌
    • Default: 22

    Port of the remote server

    # proxyHost

    • Type: string
    • Dynamic: ✔️
    • Required: ❌

    Sftp proxy host

    # proxyPassword

    • Type: string
    • Dynamic: ✔️
    • Required: ❌

    Sftp proxy password

    # proxyPort

    • Type: integer
    • Dynamic: ✔️
    • Required: ❌

    Sftp proxy port

    # proxyType

    • Type: string
    • Dynamic: ✔️
    • Required: ❌

    Sftp proxy type

    # proxyUser

    • Type: string
    • Dynamic: ✔️
    • Required: ❌

    Sftp proxy user

    # rootDir

    • Type: boolean
    • Dynamic: ✔️
    • Required: ❌
    • Default: true

    Is path is relative to root dir

    # to

    • Type: string
    • Dynamic: ✔️
    • Required: ❌

    The destination path

    # username

    • Type: string
    • Dynamic: ✔️
    • Required: ❌

    Username on the remote server

    # Outputs

    # from

    • Type: string

    The fully-qualified URIs that point to destination path

    # to

    • Type: string

    The fully-qualified URIs that point to source data
