Move 

type: "io.kestra.plugin.fs.sftp.Move"

Move a file to a SFTP server.

If the destination directory doesn't exist, it will be created

# Examples

id: "move"
type: "io.kestra.plugin.fs.sftp.Move"
host: localhost
port: 6622
username: foo
password: pass
from: "/upload/dir1/file.txt"
to: "/upload/dir2/file.txt"

# Properties

# from

  • Type: string
  • Dynamic: ✔️
  • Required: ❌

The file to move

# host

  • Type: string
  • Dynamic: ✔️
  • Required: ✔️

Hostname of the remote server

# keyfile

  • Type: string
  • Dynamic: ✔️
  • Required: ❌

Private keyfile to login on the source server with ssh

Must be the ssh key content, not a path

# passphrase

  • Type: string
  • Dynamic: ✔️
  • Required: ❌

Passphrase of the ssh key

# password

  • Type: string
  • Dynamic: ✔️
  • Required: ❌

Password on the remote server

# port

  • Type: string
  • Dynamic: ✔️
  • Required: ❌
  • Default: 22

Port of the remote server

# proxyHost

  • Type: string
  • Dynamic: ✔️
  • Required: ❌

Sftp proxy host

# proxyPassword

  • Type: string
  • Dynamic: ✔️
  • Required: ❌

Sftp proxy password

# proxyPort

  • Type: integer
  • Dynamic: ✔️
  • Required: ❌

Sftp proxy port

# proxyType

  • Type: string
  • Dynamic: ✔️
  • Required: ❌

Sftp proxy type

# proxyUser

  • Type: string
  • Dynamic: ✔️
  • Required: ❌

Sftp proxy user

# rootDir

  • Type: boolean
  • Dynamic: ✔️
  • Required: ❌
  • Default: true

Is path is relative to root dir

# to

  • Type: string
  • Dynamic: ✔️
  • Required: ❌

The destination path to move

The full destination path (with filename optionnaly)
If the destFile exists, it is deleted first.

# username

  • Type: string
  • Dynamic: ✔️
  • Required: ❌

Username on the remote server
