List 

type: "io.kestra.plugin.fs.sftp.List"

List files in a sftp server directory

# Examples

id: "list"
type: "io.kestra.plugin.fs.sftp.List"
type: io.kestra.plugin.fs.sftp.List
from: "/upload/dir1/"
regExp: ".*\/dir1\/.*\.(yaml|yml)"
