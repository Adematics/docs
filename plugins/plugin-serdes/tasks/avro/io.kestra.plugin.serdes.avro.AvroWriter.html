AvroWriter 

type: "io.kestra.plugin.serdes.avro.AvroWriter"

Read a provided file containing ion serialized data and convert it to avro.

# Inputs

# dateFormat

  • Type: string
  • Dynamic: ✔️
  • Required: ❌

Format to use when parsing date

Default value is yyyy-MM-dd[XXX].

# datetimeFormat

  • Type: string
  • Dynamic: ✔️
  • Required: ❌

Format to use when parsing datetime

Default value is yyyy-MM-dd'T'HH:mm[:ss][.SSSSSS][XXX]

# decimalSeparator

  • Type: string
  • Dynamic: ✔️
  • Required: ❌

Character to recognize as decimal point (e.g. use ‘,’ for European data).

Default value is '.'

# falseValues

  • Type: array
  • SubType: string
  • Dynamic: ✔️
  • Required: ❌

Values to consider as False

Default values are "f", "false", "disabled", "0", "off", "no", ""

# from

  • Type: string
  • Dynamic: ✔️
  • Required: ✔️

Source file URI

# nullValues

  • Type: array
  • SubType: string
  • Dynamic: ✔️
  • Required: ❌

Values to consider as null

Default values are "",
"#N/A",
"#N/A N/A",
"#NA",
"-1.#IND",
"-1.#QNAN",
"-NaN",
"1.#IND",
"1.#QNAN",
"NA",
"n/a",
"nan",
"null"

# schema

  • Type: string
  • Dynamic: ✔️
  • Required: ✔️

The avro schema associated to the data

# strictSchema

  • Type: boolean
  • Dynamic: ❌
  • Required: ❌

Whether to consider a field present in the data but not declared in the schema as an error

Default value is false

# timeFormat

  • Type: string
  • Dynamic: ✔️
  • Required: ❌

Format to use when parsing time

Default value is HH:mm[:ss][.SSSSSS][XXX]

# trueValues

  • Type: array
  • SubType: string
  • Dynamic: ✔️
  • Required: ❌

Values to consider as True

Default values are "t", "true", "enabled", "1", "on", "yes"

# Outputs

# uri

  • Type: string

URI of a temporary result file
